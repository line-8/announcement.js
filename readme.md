## Installation

- **Node.js**  
  Install the `announcement.js` package from npm

  ```
  npm i announcement.js
  ```

- **UMD**  
  Load the `announcement` module from a CDN

  ```
  https://cdn.jsdelivr.net/npm/announcement.js/lib/index.min.js
  ```
